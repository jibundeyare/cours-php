les fonctions internes (les fonctions de php)

PHP: Function Reference - Manual

Php fourni un grand nombre de fonctions. Ces fonctions permettent par exemple :

  • d’afficher un message
  • d’effectuer des opérations mathématiques
  • de comparer des variables
  • de manipuler des chaînes de caractère
  • de manipuler des tableaux
  • de débogger du code php
  • de lire ou écrire dans un fichier
  • de lire ou écrire dans une base de données
