lister les fichiers d’un répertoire

avec Windows

dir

avec Max OS X et Linux

ls

aller dans le répertoire home

avec Windows

cd /D %HOMEPATH%

avec Mac OS et Linux

cd ~

accéder à un sous-répertoire

cd [nom-du-répertoire]

accéder au répertoire parent

cd ..

se rendre dans le dossier du projet web php01.cnam.dev

Admettons que le dossier projet se nomme www.

avec Windows si le dossier www est dans la home

cd /D %HOMEPATH%
cd www/php01.cnam.dev

avec Windows si le dossier www est à la racine du disque C:

C:
cd C:/www/php01.cnam.dev

avec Mac OS et Linux

cd ~/www/php01.cnam.dev
