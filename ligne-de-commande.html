se rendre dans le dossier où se trouvent les fichiers de code php

Admettons que le dossier NFA017 se trouve sur votre desktop.

avec Mac OS X

cd ~
cd Desktop
cd NFA017
cd cours
cd code

avec Windows

cd /D %HOMEPATH%
cd Desktop
cd NFA017
cd cours
cd code

Attention : selon la langue du système, le dossier Desktop peut s’appeler Bureau.

avec Linux

cd ~
cd Desktop
cd NFA017
cd cours
cd code
