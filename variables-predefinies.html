Développement web côté serveur avec PHP

les variables prédéfinies ($_GET, $_POST, …)

PHP: Variables prédéfinies - Manual

  • $argc permet de connaître le nombre d’argument passés en ligne de commande
  • $argv permet de récupérer la valeur des arguments passés en ligne de commande
  • $_COOKIE permet de stocker et récupérer des valeurs stockées dans le cookie du navigateur
  • $_ENV permet de récupérer les valeurs des variables d’environnement
  • $_FILES permet de récupérer les données d’un fichier envoyé par un formulaire
  • $_GET permet de récupérer les données envoyées dans la barre d’adresse
  • $_POST permet de récupérer les données envoyées par un formulaire
  • $_SERVER permet de récupérer des valeurs définies par le serveur web
  • $_SESSION permet de stocker et récupérer des valeurs durant la visite d’un internaute

Toutes les variables (excepté $argc et $argv) sont des tableaux associatifs.
La variable $argv est un tableau simple contenant les arguments passés au script en ligne de commande.
La variable $argc est de type nombre entier et indique le nombre d’éléments dans la variable $argv.

En ligne de commande, seules les variables $argc, $argv, $_ENV sont utiles.
Quand php est appelé par un serveur web, toutes les variables peuvent être utilisée sauf $argc, $argv.

Note : consulter les formulaires pour voir comment récupérer les données des formulaires

code/session-set.php

 1 <?php

code/cookie-destroy2.php

 1 <?php
 2 // suppression de tous les cookies
 3 if (isset($_SERVER['HTTP_COOKIE'])) {
 4     $cookies = explode(';', $_SERVER['HTTP_COOKIE']);
 5     foreach($cookies as $cookie) {
 6         $parts = explode('=', $cookie);
 7         $name = trim($parts[0]);
 8         setcookie($name, '', time() - 1);
 9         setcookie($name, '', time() - 1, '/');
10     }
11 }

la variable $_FILES

code/file1.php

 1 <?php
 2 // ce document html contient un formulaire avec un champ nommé `fichier` permettant de pousser (uploader) un fichier vers un serveur
 3 ?><!DOCTYPE html>
 4 <html>
 5 <head>
 6 <meta charset="utf-8" />
 7 <title></title>
 8 </head>
 9 <body>
10 
11 <?php // il faut utiliser l'attribut `enctype="multipart/form-data"` pour que le fichier puisse être envoyé ?>
12 <form action="" method="post" enctype="multipart/form-data" >
13     <label for="fichier">fichier</label><br />
14     <input name="fichier" type="file" /><br />
15     <input type="submit" value="envoyer" />
16 </form>
17 
18 <?php
19 // la variable `$_FILES` est un tableau
20 // avant que l'utilisateur ne valide le formulaire la variable `$_FILES` ne contient aucune donnée
21 // quand l'utilisateur valide le formulaire, on retrouve les information concernant le fichier uploadé dans la variable `$_FILES`
22 echo '<pre>';
23 var_dump($_FILES);
24 echo '</pre>';
25 ?>
26 
27 </body>
28 </html>
