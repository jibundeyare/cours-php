Développement web côté serveur avec PHP

nom de domaines locaux (le fichier hosts)

Pour utiliser des nom de domaines locaux personnalisés, nous allons modifier le fichier hosts.

windows

Attention : vous devez ouvir votre éditeur de code en tant qu’administrateur pour pouvoir faire les modifications nécessaires.

Si vous ne savez pas comment ouvrir votre éditeur de code en tant qu’administrateur, il y a ce tutoriel (en anglais) Run a Command as Administrator from the Windows Run Box (Any Version) ou cette question / réponse (en anglais aussi) Run as administrator from command line on Windows 8 - Super User.

En tant qu’administrateur, ouvrez avec votre éditeur de code favori le fichier qui se trouve à cet emplacement :

%SystemRoot%\System32\drivers\etc\hosts

La pluart du temps, votre %SystemRoot% est C:, ce qui donne ceci :

C:\Windows\System32\drivers\etc\hosts

Ajoutez-y la ligne suivante à la fin du fichier :

192.168.33.10 php01.cnam.dev

Mac OS + Linux

Attention : vous devez ouvir votre éditeur de code en tant qu’administrateur pour pouvoir faire les modifications nécessaires.

Le plus simple est d’utiliser nano en ligne de commande :

sudo nano /etc/hosts

Avec nano, vos devez utiliser la combinaison de touches CTRL O pour enregistrer et pour sortir CTRL X.

En tant qu’administrateur, ouvrez avec votre éditeur de code favori le fichier qui se trouve à cet emplacement :

/etc/hosts

Ajoutez-y la ligne suivante à la fin du fichier :

192.168.33.10 php01.cnam.dev
