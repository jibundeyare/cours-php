vérification de l’installation

Ouvrez une nouvelle console et testez la commande suivante :

php --version

Si vous voyez un message semblable à ce qui suit, votre configuration est ordre :

PHP 5.6.0 (cli) (built: Sep 13 2014 10:36:37)
Copyright (c) 1997-2014 The PHP Group
Zend Engine v2.6.0, Copyright (c) 1998-2014 Zend Technologies

Si vous ne voyez pas ce type de message, consultez amp.html et cli.html

un premier script php en ligne de commande

php -a
echo "Hello World!\n";

Attention : sous Windows, pour faire fonctionner cette commande, il faut en plus appuyer sur les touches CTRL Z puis ENTER.

un premier script php dans un fichier

code/hello.php

pour exécuter le fichier

cd [CHEMIN DU DOSSIER]
php hello.php

en complément, voir ligne-de-commande.html pour apprendre à se servir de la ligne de commande
