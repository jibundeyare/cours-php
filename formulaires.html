Développement web côté serveur avec PHP

les formulaires

Les champs suivant ne renvoient qu’une seule valeur :

  • champ input de type text
  • champ input de type password
  • champ input de type hidden
  • champ input de type radio
  • champ input de type file
  • champ select
  • champ textarea

Les champs suivant renvoient plusieurs valeurs sous forme de tableau :

  • champ input de type checkbox
  • champ select avec l’attribut multiple="multiple"

Note : tous les champs peuvent être configurés pour renvoyer plusieurs valeurs sous forme de tableau en ajoutant des crochets à l’attribut name.

<input name="auteurs[]" type="text" value="" /><br />
<input name="auteurs[]" type="text" value="" /><br />
<input name="auteurs[]" type="text" value="" /><br />

Voir aussi :

  • les variables prédéfinies ($_GET, $_POST, $_SESSION, …)

code/form-input-file.php

 1 <?php
 2 
 3 // les champs `input` de type `file` permettent de pousser (uploader) un fichier vers un serveur
 4 
 5 $validation = true;
 6 
 7 // @note les champs `input` de type `file` ne peuvent pas avoir de valeur par défaut
 8 
 9 // les fichiers envoyés seront enregistrés dans le dossier courant
10 $upload_dir = __DIR__ ;
11 
12 // messages par défaut
13 $messages = array(
14     'fichier' => '',
15     'formulaire' => ''
16 );
17 
18 // vérification des fichiers envoyés
19 if ($_FILES) {
20     // validation du champ 'fichier'
21     if (isset($_FILES['fichier']) && !empty($_FILES['fichier'])) {
22         switch ($_FILES['fichier']['error']) {
23         case UPLOAD_ERR_OK:
24             // il n'y a pas d'erreur
25             // on peut stocker le fichier
26             if (!move_uploaded_file($_FILES['fichier']['tmp_name'], $upload_dir . '/' . $_FILES['fichier']['name'])) {
27                 // il y a une erreur
28                 // php n'est pas parvenu à stocker le fichier
29                 $validation = false;
30                 // affectation d'un message d'erreur
31                 $messages['fichier'] = "une erreur est survenue, le fichier '{$_FILES['fichier']['name']}' n'a pas pu être enregistré";
32             }
33             break;
34         case UPLOAD_ERR_NO_FILE:
35             // il y a une erreur
36             // l'utilisateur n'a pas envoyé de fichier
37             $validation = false;
38             // affectation d'un message d'erreur
39             $messages['fichier'] = "merci de sélectionner un fichier";
40             break;
41         default:
42             // il y a une erreur
43             $validation = false;
44             // affectation d'un message d'erreur
45             $messages['fichier'] = "une erreur est survenue, le fichier '{$_FILES['fichier']['name']}' n'a pas pu être enregistré";
46         }
47     }
48 
49     // vérification de la validation
50     if ($validation) {
51         // il n'y a pas d'erreurs
52         // affectation d'un message de confirmation
53         $messages['formulaire'] = "merci d'avoir rempli le formulaire";
54     } else {
55         // il y a des erreurs
56         // affectation d'un message d'erreur
57         $messages['formulaire'] = "merci de remplir le formulaire et de le valider";
58     }
59 }
