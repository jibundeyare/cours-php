Développement web côté serveur avec PHP

la portée des variables

  • PHP: Portée des variables - Manual
  • PHP: $GLOBALS - Manual

Attention : il vaut mieux éviter d’‘utiliser les mots clés global et la variable $GLOBALS car à terme ces techniques peuvent poser des problèmes de lisibilité et d’architecture du code.

Il faut noter cependant que le CMS WordPress fait un usage extensif du mot clé global. Il est donc possible d’utiliser cette fonctionnalité dans une application d’envergure.

code/portee1.php

1 <?php
2 // la variable `$a` est définie dans la portée globale
3 // une variable est toujours visible par du code se trouvant au même niveau
4 
5 $a = 'toto';
6 
7 // affiche "toto"
8 echo $a;
9 echo "\n";

code/portee2.php

 1 <?php
 2 // la variable `$a` est définie dans la portée globale
 3 // une variable déclarée dans la portée globale n'est pas visible par du code se trouvant dans une fonction
 4 
 5 $a = 'toto';
 6 
 7 function test() {
 8     echo $a;
 9     echo "\n";
10 }
11 
12 // affiche l'erreur "PHP Notice:  Undefined variable: a"
13 test();
